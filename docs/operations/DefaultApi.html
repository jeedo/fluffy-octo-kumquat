This is the API

outagesAvcGet

The Outages by AVC address end point provides information on current and completed planned and unplanned service interruptions affecting that AVC address

URL

https://api.telstra.com/v1/outages/avc

HTTP Method

GET

Response Type

Outage

Parameters

  • Query: avc String

    AVC to search by.

  • Query: content_type String

    Header to return

  • Query: callback String

    Optional wrapper for JSONP

outagesStreetAddressGet

The Outages by geographic location end point provides information on current and completed planned and unplanned service interruptions affecting that location

URL

https://api.telstra.com/v1/outages/street_address

HTTP Method

GET

Response Type

Outage

Parameters

  • Query: content_type String

    Header to return

  • Query: number String

    Address Street number.

  • Query: street String

    Street name (required if number provided)

  • Query: locality String

    Suburb or Town.

  • Query: state String

    State.

  • Query: lat BigDecimal

    Latitude.

  • Query: lpm BigDecimal

    Longitude.

  • Query: callback String

    Optional wrapper for JSONP
